# prompts/（AI用）

このディレクトリは **AIが自己参照する作業テンプレ**です。
人間がコピペして使う前提ではありません。

目的:
- 不明点の質問漏れを減らす
- 作業計画と確認手順の抜けを減らす
- 中断前に更新すべき台帳（runbook/status）を固定する

使い方（AI）:
- 作業開始時: `00_context-pack.md` を参考に、必要情報が揃っているか確認する
- 機能追加: `10_feature.md` を参考に、PRに含める内容（確認・docs更新）を漏らさない
- 中断前: `40_stop.md` をそのまま実行し、再開可能な状態で止める
