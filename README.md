# 回線運用台帳

ローカル優先・暗号化前提の PWA として、回線・期限・証跡・特典を安全に管理するためのリポジトリです。

## 現在できること
- `/` で保存済み回線の実データから危険案件、近日期限、月額合計を集計表示できる
- 危険案件が 0 件でも `問題なし` を表示できる
- `/lines` で回線ドラフトを追加・編集・削除できる
- `/lines` で回線種別、月額費用、次回確認日を保持できる
- `/lines` でキーワード、契約状態、回線種別による絞り込みができる
- `/lines` で次回確認日順、月額費用順、作成日時順の並び替えができる
- `/lines` で期限超過、今日期限、3日以内、7日以内、期限未設定を見分けられる
- `/lines` で複数選択、一括ステータス変更、一括削除ができる
- `/lines` で行ごとの詳細表示を開閉できる
- 直前の追加・更新・削除を `操作を戻す` または `Ctrl+Z` / `⌘Z` で戻せる
- `/settings` で永続ストレージ状態、schema version、保存件数、最終更新時刻、保存形式を確認できる
- `/settings` で JSON バックアップのエクスポート / インポートができる
- manifest / service worker / offline shell を確認できる

## セットアップ
```bash
npm install
npm run dev
```

## 確認手順
1. ブラウザで `/lines` を開く
2. 各行に `詳細を開く` / `詳細を閉じる` があることを確認する
3. 折りたたみ時は要点だけ見え、展開時にメモ、保存日時、期限ステータス、契約状態が見えることを確認する
4. 検索・絞り込み・並び替え・期限表示・複数選択・一括操作と併用しても期待どおり動くことを確認する
5. `/` と `/settings` の既存機能が壊れていないことを確認する
