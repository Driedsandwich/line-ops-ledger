# 回線運用台帳

ローカル優先・暗号化前提の PWA として、回線・期限・証跡・特典を安全に管理するためのリポジトリです。

## 現在できること
- `/` でダッシュボード骨組みを表示
- 危険案件が 0 件でも `問題なし` を表示
- `/lines` で回線一覧プレースホルダーを表示
- `/settings` で永続ストレージ状態を確認
- `persist を要求する` ボタンで永続化要求の結果を確認
- manifest / service worker / offline shell を確認

## セットアップ
```bash
npm install
npm run dev
```

## 確認手順
1. ブラウザで `/` を開く
2. 危険案件サマリーが `0件 / 問題なし` で表示されることを確認
3. `/settings` を開く
4. 永続ストレージ状態と容量目安が表示されることを確認
5. `persist を要求する` を押して結果メッセージが表示されることを確認
6. ブラウザの Application タブで manifest / service worker を確認
